🚀 PriorityPulse - Smart Complaint Management Application

📌 PriorityPulse is a web-based application that manages complaints through Flask and SQLite, thereby enabling the users to lodge their complaints and also helping administrators to manage them.

✨ Features:

📝 Lodging Complaints
🤖 Automatic Prioritization of Complaints (High, Medium, Low)
📊 Administering Dashboard
✅ Managing Resolution of Complaints
🗑️ Managing Complaints
💾 Managing Database

🎯 The software provides efficient and quick management of complaints.
